
# Frequently Used Linux Commands

This document summarizes the commands used during this Linux practice exercise.

## Navigation
- `pwd` — print current directory
- `ls`, `ls -la` — list files
- `cd <directory>` — move between directories

## File & Directory Management
- `mkdir` — create directories
- `touch` — create files
- `rm`, `rm -r` — remove files/directories
- `cat` — display file contents
- `nano` — edit text files

## Searching
- `find . -type f`
- `find . -name "*.txt"`
- `grep "Linux" file.txt`
- `grep -r "Linux" .`

## Processing Text
- `head`
- `tail`
- `wc`
- `sort`
- `uniq`
- `tr`

## Git Basics
- `git status`
- `git add .`
- `git commit -m "Initial Linux practice"`
- `git push`

These commands cover the core tasks completed in this repository.
