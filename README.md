# Linux Commands Project 2

## 📌 Project Overview

This project demonstrates practical usage of essential Linux commands by performing file and directory management, text processing, searching, and Git version control operations. The project was completed using **Kali Linux** running on **VMware Workstation Pro**.

---

## 🎯 Objectives

- Learn basic Linux terminal commands.
- Practice file and directory management.
- Perform file searching and text processing.
- Understand Linux command-line workflow.
- Learn Git and GitHub version control.

---

## 🛠️ Technologies Used

- Kali Linux
- VMware Workstation Pro
- Visual Studio Code
- Git
- GitHub
