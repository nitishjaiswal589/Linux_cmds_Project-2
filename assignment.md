
# Linux Practice Assignment

## Goal

Become comfortable with everyday Linux terminal commands by creating a small directory structure and performing basic file operations.

## Activities

1. Create separate folders for HR, IT, Sales and Finance.
2. Add multiple sample employee files inside each folder.
3. Insert simple text into the created files.
4. Practice commands including:
   - find
   - grep
   - sort
   - uniq
   - wc
5. Verify results from the terminal.
